## How to Use

1. Open the `index.html` file in any modern web browser
2. Enter student information (name, class, roll number)
3. Add subjects using the "Add Subject" button
4. Enter marks for each subject (0-100)
5. Click "Calculate Grade" to see results
6. View the detailed result summary
7. Use control buttons to reset, print, or download results

## Grade Scale

The calculator uses the following grade scale:
- A+: 90% and above (Excellent)
- A: 80-89% (Very Good)
- B: 70-79% (Good)
- C: 60-69% (Satisfactory)
- F: Below 60% (Fail)

## Result Summary

The application displays:
- Total marks across all subjects
- Average percentage
- Grade point (simplified calculation)
- Final letter grade with color coding
- Grade description
- Number of subjects
