# CODSOFT Internship Tasks - Web Applications

Welcome to my CODSOFT internship task repository. This repository contains solutions to various programming tasks assigned during my internship, now enhanced as modern web applications.

## Tasks Completed

1. **Task 1: Number Guessing Game (Web Application)**
   - Interactive number guessing game with adjustable difficulty levels
   - Features scoring system, guess history, and progress tracking
   - Modern UI with dark/light mode and smooth animations
   - Built with HTML, CSS, and JavaScript

2. **Task 2: Student Grade Calculator (Web Application)**
   - Calculates total marks, average percentage, and assigns letter grades
   - Supports multiple subjects with dynamic addition/removal
   - Features print/download functionality and student information tracking
   - Built with HTML, CSS, and JavaScript

3. **Task 3: Currency Converter (Web Application)**
   - Real-time currency converter with API integration for 200+ currencies
   - Features conversion history, swap functionality, and live rates
   - Modern UI with dark/light mode and responsive design
   - Built with HTML, CSS, and JavaScript

## Technologies Used

- HTML5
- CSS3 (with modern features like Flexbox, Grid, and CSS variables)
- JavaScript (ES6+)
- External APIs (for currency converter)

## How to Run

Each task is contained in its own directory with an `index.html` file. To run any task:

```bash
# Open the index.html file in any modern web browser
# Or serve using a local server
cd [task-directory]
python3 -m http.server 8000
```

Then open your browser to `http://localhost:8000`
